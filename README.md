معرفی بازی “نقطه‌زن”

در بازی نقطه‌زن، شما نقش یک تیرانداز حرفه‌ای را دارید که باید دشمنان را قبل از رسیدن به پایین صفحه از بین ببرد.

ویژگی‌ها:

دشمنان با حرکت زیگزاگی ظاهر می‌شوند و شلیک آن‌ها را باید با دقت خنثی کنید.

بازی دارای مراحل مختلف است که با کسب امتیاز، سخت‌تر می‌شوند.

جان شما محدود است و باید با دقت تیراندازی کنید تا زنده بمانید.

Crosshair دایره‌ای و نورانی محل تیراندازی شما را نشان می‌دهد و هنگام شلیک افکت جذابی دارد.

بازی بدون آیتم اضافی است و تمرکز بر مهارت تیراندازی و سرعت عمل شما می‌باشد.

هدف شما این است که تا آخرین مرحله زنده بمانید و بیشترین امتیاز ممکن را کسب کنید.
